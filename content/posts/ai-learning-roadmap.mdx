## 学习路线全景图

```
阶段一          阶段二          阶段三          阶段四          阶段五          阶段六
入门开发  →   应用开发   →   核心开发   →   智能体开发  →   定制开发   →   算法进阶
(初级)        (数据分析)      (ML/DL)       (Agent/RAG)    (微调/推理)    (CV/多模态)
```

## 阶段一：AI 大模型开发入门

**目标：** 能用 Python 调用大模型 API，搭建简单的聊天机器人。

| # | 学习内容 | 天数 | 推荐资源 |
|---|---------|------|---------|
| 1 | 大模型基础概念、Ollama 本地部署 | 1 | [Ollama 官方文档](https://ollama.com)、[ChatBox 下载](https://chatboxai.app) |
| 2 | Python 基础语法（变量、控制流、容器） | 3 | [Python官方教程](https://docs.python.org/zh-cn/3/tutorial/)、[廖雪峰Python教程](https://www.liaoxuefeng.com/wiki/1016959663602400) |
| 3 | Python 函数、文件操作、异常处理 | 2 | 同上 + [realpython.com](https://realpython.com) |
| 4 | Streamlit + API 调用搭建聊天机器人 | 2 | [Streamlit 官方文档](https://docs.streamlit.io)、[通义千问API](https://help.aliyun.com/zh/model-studio/) |

**阶段项目：** 用 Streamlit + 通义千问 API 搭建一个 Web 聊天页面。

## 自学建议

```
┌────────────────────────────────────────────────────────────┐
│  阶段一 → 阶段二：打基础，不要跳                              │
│  ─────────────────────────────────────                      │
│  必须掌握：Python + SQL + Pandas + 提示词                     │
│  检验标准：能独立用 Streamlit + API 搭一个聊天应用             │
├────────────────────────────────────────────────────────────┤
│  阶段三：最关键的转折点                                       │
│  ─────────────────────────                                  │
│  必须掌握：ML 基础 + PyTorch + Transformer 原理               │
│  检验标准：能看懂 BERT 论文并跑通 HuggingFace 微调             │
├────────────────────────────────────────────────────────────┤
│  阶段四 → 阶段五：深水区                                      │
│  ─────────────────────────                                  │
│  必须掌握：LangChain + RAG + 微调 + 推理优化                  │
│  检验标准：能从零搭建 RAG 系统并完成模型微调部署               │
├────────────────────────────────────────────────────────────┤
│  阶段六：选学                                                 │
│  ────────────                                                │
│  CV/多模态是加分项，不是必选项                                │
│  根据目标岗位决定是否深入                                     │
└────────────────────────────────────────────────────────────┘
```

**关键原则：**
1. **先跑通再理解** — 不要死磕理论，先 clone 项目跑起来
2. **每个阶段做一个完整的项目** — 项目是检验学习的唯一标准
3. **善用免费 GPU** — Google Colab、Kaggle、AutoDL 都有免费额度
4. **加入社区** — [Hugging Face Discord](https://discord.gg/hugging-face)、[飞浆 AI Studio](https://aistudio.baidu.com)
