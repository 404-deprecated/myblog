## 一、AI市场规模：天花板在哪？

全球AI市场2026年已达**5145亿美元**，预计到2032年超过1.3万亿美元，复合增长率约25%。生成式AI细分市场2026年达到916亿美元，到2030年预计突破4000亿美元，年均增长34%。

**四层产业结构，赚钱方式完全不同：**

| 层次 | 代表公司 | 商业模式 |
|------|---------|---------|
| 算力/芯片层 | 英伟达、AMD | 卖铲子，最确定 |
| 基础设施/云层 | 台积电、AWS、CoreWeave | 建设+收租 |
| 模型层 | OpenAI、Anthropic | 订阅+API调用 |
| 应用层 | Cursor、各类垂直SaaS | 最多机会，最快迭代 |

## 二、大公司的护城河重构

AI时代，**速度不再是护城河**——当AI让所有人都能快速出货，速度就失去了差异化价值。真正能活下来的护城河是：

### 护城河1：专有数据飞轮

真正的数据优势由三层构成：竞争对手无法复制的专有数据 + 历史积累形成的"机构记忆" + 持续改进的反馈循环。三者叠加，系统越用越强，壁垒越来越宽。

微信有14亿用户社交图谱，美团有数亿人的外卖偏好——这些数据根本买不到。

### 护城河2：系统级嵌入（切换成本）

SAP和Oracle的触手深入大量企业，很多公司试图迁移都以失败告终。如果SAP宕机，工厂停产；如果Workday崩溃，工资就发不出去。AI是在增强这些系统，而不是替代它们。

### 护城河3：分发网络

强大的分发渠道可以击败强大的算法，尤其是当算法本身可以被复制的时候。微软把Copilot塞进Office、Teams、VS Code——这不是产品力，是分发力。

### 护城河4：物理基础设施

英伟达的CUDA生态、苹果的自研Neural Engine芯片、华为的昇腾软件栈——这些是纯靠钱和时间砸出来的，代码追不上的。

### 什么不再是护城河

单纯靠功能堆叠的SaaS公司正在被重新定价。AI正在降低换供应商的摩擦，靠"绑架客户而不是服务客户"的公司面临前所未有的压力。

## 三、大公司裁员：正在发生

2026年仅到4月中，科技行业已裁员超过**15万人**，是近十年来最集中的一波裁员潮。Amazon裁减约1.6万企业岗位，Oracle裁减约3万人（约20%员工），Meta裁减8000人。

**被裁的岗位：**
- 客服、质量保证、内容审核
- 中层管理
- 初级分析师、初级程序员

**与此同时严重短缺的岗位：**
- 机器学习工程师
- AI安全研究员
- 数据基础架构专家
- AI相关岗位招聘量2026年增长92%，薪资溢价高达56%

世界经济论坛预测到2030年，22%的岗位将受到冲击，但同时将创造1.7亿个新岗位，净增约7800万个就业机会。

### 未来架构模式

```
传统模式：100人公司 → 做100人的事
未来模式：10人 + AI Agent → 做1000人的事
```

- **"小核心+大外包AI"结构**：公司极少数战略人才保留，执行层大量用AI Agent替代

- **中间层消失**：中层管理、初级分析师最先被压缩
- **Agent经济**：公司买的不再是软件，而是"能力"——按任务完成数量付费

## 六、供应链依赖图谱

三家核心公司看似独立，实际上共享同两个最关键节点：**台积电**和**SK Hynix**。

### 英伟达的五层依赖

```
英伟达
├── 晶圆制造：台积电 3nm/4nm（主）+ 三星（备用）
├── 先进封装：台积电 CoWoS（包揽全球60%产能）
│             ASE/SPIL + Amkor（备用OSAT）
├── 高带宽内存：SK Hynix HBM4（优先供货）
│             Samsung HBM3e（40%份额）
│             Micron（美国备选）
├── 网络互联：Arista交换机 + 中际旭创光模块
└── 软件生态：CUDA（自有，唯一不依赖他人的护城河）
```

**致命风险**：台积电CoWoS产能至少被英伟达包揽60%，产能至2027年全部锁定，台积电断供=停产。

### Broadcom的依赖

```
Broadcom
├── 晶圆制造：台积电 3nm（谷歌TPU专用产线）
├── 产能预订：已锁定至2028年
├── HBM内存：SK Hynix + Samsung
└── 新瓶颈（2026年出现）：激光器组件 + PCB基板
```

Broadcom CEO已公开表示：TSMC容量是2026年最大瓶颈，且PCB和激光器意外成为新卡点。

### 海光信息的三重约束

```
海光信息
├── 架构授权：AMD Zen一代（2016年永久授权，仅第一代，不含后续）
├── 晶圆制造：中芯国际 N+2 7nm（良率是核心瓶颈）
│             台积电南京工厂（受美国芯片法案限制扩产）
├── EDA工具：Synopsys + Cadence（部分尚未完成续约）
└── 封测：国内封测厂（依赖度低，可替代性强）
```

**最大风险**：EDA工具断供=无法出新产品。三家公司中地缘政治脆弱性最高的。

## 九、最重要的结构性判断

麦肯锡估计2030年前AI基础设施总投资需要**5.2万亿美元**。芯片只是其中一部分，实体数据中心、电网、散热系统同样需要大规模投入。

**最容易被忽视、但逻辑最硬的两家公司：**

- **Vertiv**——不管哪家芯片公司赢，数据中心都要散热
- **台积电**——不管谁设计芯片，最先进的都只能在台积电造

这两家是AI基建里真正的"铲子卖家"，风险比英伟达小，上涨空间仍然很大。

**中国侧最重要的结构性逻辑：**

国产替代是政策驱动的刚需，不是纯市场选择。中国AI芯片市场将形成以华为+寒武纪为核心的"双头格局"，全球AI产业链正在加速分裂，中美最终将形成完全独立的AI生态系统。

在这个框架下，**中芯国际和北方华创是风险收益比最好的标的**——前者是不可替代的制造基础设施，后者吃的是"美国制裁竞争对手+中国建厂需求爆发"的双重红利。
