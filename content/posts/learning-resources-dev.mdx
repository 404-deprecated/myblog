## Android

### 官方文档

- [Android Developers](https://developer.android.com) — 遇到 API 问题首选，比任何第三方教程都准确
- [Now in Android](https://github.com/android/nowinandroid) — Google 官方示例 App，Architecture + Compose 最佳实践的集中展示
- [Jetpack Compose Samples](https://github.com/android/compose-samples) — 各类 Compose 组件的参考实现

### 值得读的博客

- [Chris Banes's blog](https://chrisbanes.me) — Compose、WindowInsets 方向，深入
- [Florina Muntenescu](https://medium.com/@florina.muntenescu) — Architecture Components 系列文章

### 书

- 《Kotlin in Action》— Kotlin 语法入门，比官方文档读起来更流畅

---

## iOS / 跨平台

### Swift / SwiftUI

- [Swift by Sundell](https://www.swiftbysundell.com) — 文章质量高，不水
- [Hacking with Swift](https://www.hackingwithswift.com) — 项目驱动学习，入门友好

### Flutter

- [Flutter官方文档](https://docs.flutter.dev) — 比 Android 官方文档写得更好读
- [Reso Coder](https://resocoder.com) — 架构向，Clean Architecture + BLoC 系列很系统

## 通用工具

| 工具 | 用途 |
|------|------|
| [Excalidraw](https://excalidraw.com) | 快速画架构图、流程图 |
| [Ray.so](https://ray.so) | 代码截图，比截屏好看 |
| [Regex101](https://regex101.com) | 调试正则表达式 |
| [cURL to code](https://curlconverter.com) | 把 cURL 命令转成各语言代码 |
| [TLDraw](https://www.tldraw.com) | 白板工具，画草图 |

## 保持信息更新

我主要通过这几个渠道跟进技术动态：

1. **GitHub Trending** — 每周看一次，了解社区在关注什么
2. **Android Weekly newsletter** — 每周邮件，筛选质量不错
3. **Twitter/X** — 关注几个核心开发者，噪音多但信噪比还行
