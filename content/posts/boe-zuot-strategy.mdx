## 为什么是京东方A？

我在持仓和十五五策略池中，对所有A股做了30日振幅回溯分析，京东方A碾压式胜出：

| 指标 | 京东方A | 三花智控 | 隆基绿能 | 招商银行 |
|---|---|---|---|---|
| 股价 | **¥6.15** | ¥47 | ¥13 | ¥38 |
| 一手成本 | **¥615** | ¥4,701 | ¥1,334 | ¥3,810 |
| 30日日均振幅 | **4.4%** | 4.7% | 2.8% | 1.1% |
| 振幅>2%天数占比 | **70%** | 93% | 73% | 7% |
| 日成交额 | **200亿** | 78亿 | 14亿 | 28亿 |
| 做T评分 | **95** 🥇 | 75 🥈 | 65 | 65 |

**三花智控振幅更高（4.7%），但京东方A胜在三点：**

1. **价格极致低** — ¥6.15一股，一手615元。10万可以买16,000股（162手），做T时2,000-3,000股的买卖不影响盘口
2. **流动性极致好** — 日成交200亿，是A股流动性最好的股票之一。市价单秒成交，不会因为你的2000股导致滑点
3. **周期底部的安全边际** — PB ~1.0 倍（净资产附近），面板价格处于周期底部，下行空间有限

## Serenity视角：京东方A在面板产业链的位置

### 瓶颈点分析

京东方A 不是传统 Serenity 意义上的「紫苏叶」。面板行业竞争激烈——三星显示、LG Display、华星光电（TCL科技）、惠科都在做。没有独家垄断，没有定价权。

**但 Serenity 方法论里有一个隐含前提：你不需要垄断整个产业链，你只需要垄断「散户做T」这个场景。**

京东方A 在这个场景里的垄断地位：
- ¥6的价格排除机构大资金（机构不做¥6以下的股票）
- 200亿日成交排除流动性风险
- 面板行业的周期性提供天然波动
- 15年A股历史积累了海量散户共识

**这就是「做T场景下的紫苏叶」——不是技术垄断，而是交易属性垄断。**

### 长期价值：玻璃基载板是第二增长曲线

2026年6月3日机构调研纪要披露了一句话引爆涨停：**「有客户预计2028年玻璃基载板渗透率达30%」**。

玻璃基载板（Glass Core Substrate）是AI芯片先进封装的关键材料，替代传统有机基板。京东方在玻璃加工领域有深厚积累（LCD面板本身就是玻璃基板技术），这个技术延伸到半导体封装是顺理成章的。

- 面板主业：周期底部，LCD价格企稳回升。PB 1.0 倍的历史底部区域
- 玻璃基载板：如果2028年渗透率真到30%，这是一个千亿级新市场
- OLED：成都、绵阳产线逐步满产，折叠屏渗透率提升

> 长线逻辑：面板周期反转 + 玻璃基载板期权 = 下行有限，上行可期

## 10万本金做T策略

### 第一步：建立底仓

| 参数 | 配置 |
|---|---|
| 总资金 | **¥100,000** |
| 底仓 | **¥61,500**（10,000股 × ¥6.15） |
| 做T资金 | **¥30,000**（每日做T使用） |
| 现金储备 | **¥8,500**（应对极端情况） |

**底仓为什么是10,000股？**
- 10,000股是整数，做T时容易计算
- 每1%波动 = ¥615，心理上容易接受
- 剩余3万做T足够覆盖每日2,000-3,000股的买卖

### 第二步：每日开盘判断

```
09:15-09:25 集合竞价 → 判断当日方向

┌─────────────────────────────────────────────────────┐
│ 高开 >2%  → 开盘先卖 2,000-3,000 股（锁利）           │
│             等回落后买回（通常涨停次日常见）            │
│                                                     │
│ 平开 ±1%  → 不操作，等方向明确                        │
│             10:00后看趋势决定                          │
│                                                     │
│ 低开 >1%  → 开盘先买 2,000 股（抄底）                  │
│             反弹 1.5% 后卖出                           │
│                                                     │
│ 低开 >3%  → 这是黄金坑！买 3,000 股                    │
│             等日内修复反弹 2-3% 后卖出                  │
└─────────────────────────────────────────────────────┘
```

### 第三步：盘中做T触发点

**买入触发（分两批）：**
- 日内从高点回落 2% → 买 1,000 股
- 日内触及 MA5（今日约 ¥5.54）→ 买 1,000 股
- 如果上午没有触发，下午 14:00 后不再新开

**卖出触发（分两批）：**
- 日内从低点上涨 2% → 卖 1,000 股
- 日内触及昨日收盘价 +2% → 卖 1,000 股
- 尾盘 14:50 前必须平掉所有做T仓位（不留过夜）

### 第四步：盈亏纪律

| 规则 | 数值 |
|---|---|
| 单笔做T目标盈利 | 1.5-3% |
| 单笔做T止损 | -1.5%（触及立刻平仓） |
| 日内最大做T仓位 | 3,000 股（底仓的30%） |
| 连续3笔亏损 | 当日停止做T（情绪坏了） |
| 月做T目标 | 10-15笔，月增厚收益 3-5% |

### 第五步：底仓保护

**底仓不卖的核心前提：面板周期处于底部区域。**

| 底仓卖出触发 | 条件 |
|---|---|
| 京东方A跌破 ¥3.79（历史最低）| 清仓，等更低点重建 |
| 面板价格连续3月下跌 | 减仓至5,000股 |
| 玻璃基载板客户流失 | 减仓50% |
| 公司突发重大利空 | 盘中减仓 |

**只要PB还在1.0 倍附近，面板价格没有暴跌，底仓就拿着不动。**

## 风险提示

1. **不是每天都能做T。** 振幅不到1.5%的日子（约30%的交易日）不强行操作，亏手续费不值得
2. **涨停日不做T。** 今天如果涨停封板了，不要试图在板上做差价——卖飞了买不回来
3. **止损纪律是生命线。** 做T变补仓、补仓变满仓、满仓变套牢——这是散户最常见的死亡螺旋
4. **面板行业的周期性。** 如果LCD价格再次大跌，底仓的账面亏损可能超过做T的收益
5. **10万元不是大资金。** 不要试图每天满仓做T。每日3,000股的做T仓位（3成底仓）是合理的风险敞口
